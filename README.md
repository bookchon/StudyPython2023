## 2일차
1. 파이썬 기본
    - 변수
    - 자료형
    - 연산자

 ```
 pi = 3.141592
 print(f'파이는 {pi} 입니다.') # 3.141592 다 나옴
 print(f'파이는 {pi:0.2f} 입니다.') # 3.141592 다 나옴 # 0.02f = 소수점 두 번째 까지 출력하라는 뜻
 print(f'파이는 {pi:10.3f} 입니다.') # 소수점 셋째짜리까지 표현하고, 앞의 공간을 10칸을 만들라는 뜻
```
